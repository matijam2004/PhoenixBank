GENERAL SETUP:
1. pull from v1.1
2. create .env file under bank-app folder
    contents:
    VITE_API_URL=http://127.0.0.1:8000/api
    MONGO_URI=mongodb+srv://<your_username>:<your_password>@cluster0.sr7kzwx.mongodb.net/
    DB_NAME=pheonix

RUNNING FRONTEND:
1. navigate to bank-app/frontend
2. install node (I forgot the command)
3. start frontend server: npm run dev
4. troubleshoot or install necessary modules if required
5. visit http://localhost:5173/ to view webapp

RUNNING BACKEND: (in a new terminal)
1. navigate to bank-app/backend
2. install requirements.txt: pip install -r requirements.txt
3. start backend server: python3 -m uvicorn main:app --reload
4. troubleshoot or install necessary modules if required
5. visit http://127.0.0.1:8000/docs to view existing APIs
